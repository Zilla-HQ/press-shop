# Press — Meta ads, ready to launch

Three ad sets, each targeting a different reason someone walks in the door of a custom print shop. Same Meta Ads Manager setup as Anchor; the audiences and angles are different because Press isn't a holiday-coded gift play — it's a year-round utility brand.

**Launch window**: open-ended. Press has no forced deadline like Father's Day. Start whenever the domain + payments + test purchase are verified.
**Storefront**: https://pressprint.xyz

---

## Budget + targets

| | |
|---|---|
| Total daily budget | $45–60/day across 3 ad sets ($15–20 each) for first week |
| ROAS target (cold) | 1.8–2.5× — custom print has lower urgency than gift-coded, so first-touch ROAS often lags Anchor by ~10% |
| Expected CPM (cold US) | $12–22 (lower than apparel-gift CPMs because audiences are more dispersed) |
| Expected CTR (cold) | 1.2%+ on a working creative |
| Add-to-cart rate | 3%+ on the product page is the bar (lower than gift-coded because there's a design-upload step that filters casual browsers) |
| Site conversion rate | 0.8–1.5% — lower than Anchor because customization adds friction |

Press margins per unit are similar to Anchor ($12–25 each), so the $1K target = ~50 orders.

**Headline**: `Bring the design.`

**Description**: `Made-to-order. From $28.`

**CTA**: `Learn More`

**URL**: `https://pressprint.xyz/`

**Why it works**: Designers already have inventory (their unused sketches) and want to print them. The "no minimums / no setup fees" line directly addresses what makes Vistaprint/Spreadshirt feel hostile. "We don't improve your art" signals respect for the designer's taste — that's the conversion lever.

## Launch sequence

| Day | Action |
|---|---|
| **Day 0** | Launch all 3 ad sets at $15–20/day. Conversion event: `Purchase`. Lowest-cost bid. Daily budget. |
| **Day 1–2** | Don't touch. Learning phase. |
| **Day 3** | Pause anything under 0.8× ROAS. Reallocate. |
| **Day 5** | Refresh creative on whichever ad is fatiguing. Custom-merch ads fatigue slightly faster than gift ads — swap image every 5–7 days. |
| **Day 7** | First retargeting layer: warm visitors who hit `/products/<id>` but didn't buy. Show them the SAME product they viewed plus a "first order, 10% off" code (you'll need to create the code in Shopify Discounts manually). |
| **Day 14** | Lookalike audiences ON (1% of buyers). |
| **Week 4+** | If ROAS holds, layer in Dynamic Product Ads using the Shopify catalog feed. |

## What to watch

- **CPM > $25 cold**: usually a creative issue. Designer images (especially Procreate canvas / Figma frames) tend to get the lowest CPMs because Meta's algorithm classifies them as "tools," not "ads."
- **CTR < 1%**: try the second-tier audience (Etsy sellers, Discord users) rather than rewriting copy.
- **Add-to-cart < 2%**: design upload step might be the friction. Watch the funnel: is the bottleneck at "Choose file" or at "Buy"?
- **Conversion < 0.5%**: probably a domain trust issue (`.xyz` cuts CTR ~20% lower than `.com` averages). Consider buying a `.com` mirror and 301-redirecting once you have data.
